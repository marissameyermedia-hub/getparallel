# Parallel Ad Generator

Use this skill whenever creating new social media ad designs for Parallel.

## The Template
Canva template: https://www.canva.com/design/DAHKmK0J6Ls/7CewZjan-fLtdlViH45EkA/edit

## Rules — Never Break These
- **Only change**: the copy (words) and the background color
- **Never change**: fonts, font sizes, element positions, logo, URL, layout
- Logo (PARA//EL.) stays top left on every slide
- URL (getparallel.vip) stays bottom right on every slide
- Realign text neatly after changing copy
- **Text color rule**: On a purple background, all body text must be white. On black background, all body text must be white. On white background, all body text must be black.

## Background Color Pattern
Rotate in this order, always picking up where the last published post left off:
**Purple → Black → White → Purple → Black → White → ...**

Check the social planner to find the last published post's background color, then use the next color in the sequence.

## Slide Structure (3 slides per post)
- Slide 1 (cover): Short punchy hook — the problem statement
- Slide 2: Deepen the pain / explain why it happens
- Slide 3: Parallel's quiet, confident solution — no hard sell

## Workflow
1. Identify the next background color from the pattern
2. Open the Canva template and duplicate it
3. Replace copy on all 3 slides — do not change anything else
4. Apply the correct background color to all slides
5. Confirm logo top left + URL bottom right on every slide
6. Send the Canva link to the user for approval
7. Once approved, upload the post to the next open slot in the social planner dashboard at the correct scheduled time

## Copy Format
Copy is always provided in this format before invoking this skill:
- Slide 1: [cover line]
- Slide 2: [body — 2-4 sentences]
- Slide 3: [resolution — 1-2 sentences]
