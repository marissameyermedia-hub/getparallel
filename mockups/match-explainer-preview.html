Design notes

  • Trigger: Score pill on the match card (already visible, no new surface). Tap reveals the sheet.
  • AI copy (1 sentence): Claude Haiku generates the headline summary from questionnaire data only — no message content. Cached per match pair, logs to ai_cost_log.
  • Breakdown bars: Existing data from the breakdown field — no new API calls. Bars animate in on mount.
  • "Why you matched" bullets: Existing why_you_matched array from the matches table.
  • "Things to talk about": Existing potential_differences array, reframed positively.
  • Shared hobbies: Existing shared_hobbies array.
  • Flag: feature_match_explainer_enabled — sheet is suppressed when off; score pill still shows but is non-interactive.
